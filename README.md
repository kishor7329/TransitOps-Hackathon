# TransitOps-Hackathon

Smart Transport Operations Platform developed for a hackathon.

## Tech Stack

- Next.js
- Tailwind CSS

## Getting Started

1. Navigate to the `nextjs` folder.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```
